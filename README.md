# Smart Visitor Counter Berbasis Arduino

## Deskripsi Proyek

Smart Visitor Counter merupakan sistem penghitung pengunjung otomatis yang dikembangkan menggunakan platform Arduino Uno dengan memanfaatkan sensor Passive Infrared (PIR) sebagai pendeteksi pergerakan manusia dan LCD I2C sebagai media tampilan informasi. Sistem ini dirancang untuk melakukan pencatatan jumlah pengunjung yang masuk dan keluar secara otomatis serta menampilkan total pengunjung yang berada di dalam area pemantauan secara real-time.

Pengembangan proyek ini bertujuan untuk mengimplementasikan konsep Internet of Things (IoT) dan sistem embedded dalam bidang monitoring dan otomasi. Dengan adanya sistem ini, proses pendataan pengunjung dapat dilakukan secara lebih efisien, akurat, dan minim kesalahan dibandingkan dengan metode pencatatan manual.

Proyek ini dapat diterapkan pada berbagai lingkungan seperti ruang kelas, laboratorium, perpustakaan, toko, kantor, maupun area publik lainnya yang memerlukan monitoring jumlah pengunjung secara otomatis.

## Tujuan

- Mengimplementasikan teknologi Arduino dalam sistem monitoring pengunjung.
- Mengembangkan sistem penghitung pengunjung secara otomatis dan real-time.
- Meningkatkan efisiensi proses pencatatan data pengunjung.
- Menerapkan konsep Internet of Things (IoT) dan Embedded System dalam studi kasus nyata.

## Fitur Sistem

- Deteksi pengunjung masuk secara otomatis.
- Deteksi pengunjung keluar secara otomatis.
- Perhitungan total pengunjung secara real-time.
- Tampilan data pada LCD I2C 16x2.
- Sistem sederhana, ekonomis, dan mudah dikembangkan.

## Teknologi yang Digunakan

- Arduino Uno
- Sensor PIR HC-SR501
- LCD 16x2 I2C
- Bahasa Pemrograman Arduino (C/C++)
- Embedded System
